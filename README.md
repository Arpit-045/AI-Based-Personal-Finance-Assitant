# AI-Based-Personal-Finance-Assitant
Managing personal finances effectively remains a significant challenge for a large portion of the population due to lack of financial literacy, complex financial instruments, and the absence of personalized advisory tools. This project presents the design and development of an AI-Based Personal Finance Assistant, a web-based intelligent system that leverages machine learning and natural language processing to help users track their expenses, analyse spending patterns, and receive personalized financial recommendations in real time.
The primary objective of this system is to automate expense categorization, provide predictive insights about future expenditures, and deliver context-aware financial advice tailored to individual user profiles. The system integrates a conversational AI chatbot powered by a fine-tuned large language model that enables users to interact with their financial data through natural language queries. Key modules include income and expense tracking, budget planning, investment suggestions, and alert notifications for anomalous spending behaviour.
The methodology involves the use of Python and Flask for the backend, React.js for the frontend, and a combination of supervised learning models (Random Forest, LSTM) for expense prediction. The AI chatbot is powered by the GPT-based API integrated with a Retrieval-Augmented Generation (RAG) pipeline to ensure contextually accurate and personalized responses.
Results demonstrate that the system achieves an expense categorization accuracy of 92.4%, a budget prediction Mean Absolute Error (MAE) of Rs. 312, and user satisfaction scores averaging 4.3/5.0 in usability testing. The AI assistant significantly improves financial awareness and helps users make informed decisions, making it a promising tool for personal financial management.
